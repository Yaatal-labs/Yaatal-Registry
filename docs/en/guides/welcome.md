# Welcome to the YAATAL Control Center

YAATAL is a high-performance, edge-optimized livestream translation platform designed to connect speech and text across languages in real-time. This portal acts as a Single Source of Truth (SSOT) for our architecture, documentation, and workflow pipelines.

## Platform Pillars

1. **Real-time Subtitles (Caption Lane)**
   - Streaming Speech-to-Text via ASR models.
   - Zero-latency context windows for auto-punctuation.
   
2. **Audio Dubbing (Dub Lane)**
   - Autoregressive speech synthesis on edge.
   - Low-literacy accessibility mappings.

3. **Hybrid Edge Processing**
   - Local dictionary lookups running directly on device CPU.
   - WebAssembly-based translation components (ONNX).
   
## Getting Started

To explore the architecture, read the **YAATAL Livestream Lane** analysis in the sidebar. You can toggle between English and French at any time using the global language selector.
